Magic Builder

Com este plugin você pode criar magias com apenas um clique na Godot. Ele é desenvolvido para devs indie que buscam um pouco mais de facilidade na hora de criar.

Sobre o Plugin

O Magic Builder Um plugin leve e intuitivo para Godot, criado para acelerar o desenvolvimento de sistemas de magia e habilidades. Ele permite criar cenas, comportamentos e configurações avançadas de forma rápida, sem precisar escrever código repetitivo. Ideal para jogos 2D de ação, RPG, metroidvania e qualquer projeto que use habilidades personalizadas.

Atualmente, o plugin está em Beta (V 0.5) mas já conta com a base funcionando, nele você pode criar magias no modelo de projéteis, com nomes e elementos pré-configurados. No começo do mês de dezembro planejo lançar a versão 0.6, com mais dois modelos de mágias e sistema de nome personalizado.
